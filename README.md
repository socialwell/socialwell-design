# This is a socialwell web design system

## Installation

```bash
yarn

or

npm install
```

## To run the project

```bash
yarn storybook
```

or

```bash
npm storybook
```

## To build the storybook into static files

```bash
yarn build-storybook
```

or

```bash
npm build-storybook
```

## we have lint installed in the project to run the lint

```bash
yarn lint

or

yarn lint:fix
```
