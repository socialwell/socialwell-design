# This is a socialwell web design system

## Installation

```bash
yarn add @socialwell-design/alert-dialog

or

npm i @socialwell-design/alert-dialog
```

## How to use

```JSX
import { AlertDialog } from "@socialwell-design/alert-dialog";

<AlertDialog
  onClick={() => {}}
  text="Lorem ipsum dolor sit amet?"
  title="Delete user"
/>
```
