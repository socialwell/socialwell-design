# This is a socialwell web design system

## Installation

```bash
yarn add @socialwell-design/toast

or

npm i @socialwell-design/toast
```

## How to use

```JSX
import { Toast } from "@socialwell-design/toast";

<Toast
  description="This is a description"
  onClose={() => {}}
  state="success"
  title="Title"
/>

```
