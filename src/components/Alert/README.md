# This is a socialwell web design system

## Installation

```bash
yarn add @socialwell-design/alert

or

npm i @socialwell-design/alert
```

## How to use

```JSX
import { Alert } from "@socialwell-design/alert";

<Alert
  description="This is a description"
  onClose={() => {}}
  state="success"
  title="Title"
/>
```
